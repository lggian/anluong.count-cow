# Cow-counting system using YOLO + ByteTrack for livestock monitoring in the stable and at the gate for demo presale project. 

Have been approved by the business.

--------------------------------------------------------------------------------------------------------------------
How to run this:
1. Clone this repo
```bash
git clone https://github.com/lggian/anluong.count-cow.git
```
2. Import requirements.txt
```bash
pip install -r requirements.txt
```
3.1 For counting in the stable
```bash
python count_cow_stable2.py
```
3.2 For counting at the gate
```bash
python count_cow_gate.py
```
